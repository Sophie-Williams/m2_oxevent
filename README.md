# Metin2 OX Event

## Install
Enter your locale quest folder and run `./qc ox_event.lua`

## Event Flag
To run the OX event automatically set the ox_event game flag to 1.
`/e ox_event 1`

## Notes

Do not translate the weekdays in the settings!
